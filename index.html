SINTAXIS CSS

Selector

Descripción

Estos nos indican a que elemento o elementos se les aplicaran las propiedades css que definamos. Hay varias formas de seleccionar elementos. Como lo son el selector de tipo, el selector de clase, por ID, selector universal, combinador de descendientes. Tambien es importante destacar que existen Pseudoclases tales como hover, visited, entre otras. Para ver cada uno de estor ir a la seccion de selector en el menu.

Selector Universal

Descripción

La sintaxis de esta es poner un asterisco. De esta forma seleccionamos todos los elementos.

Selector de Tipo

Descripción

De esta forma seleccionamos todos los elementos que tengan esta etiqueta. En este ejemplo estariamos seleccionando todos los elementos que tengan la etiqueta button.

Selector por Id

Descripción

La sintaxis de esta es poner un simbolo del gato o tambien conocido como numeral antes del id. De esta forma seleccionamos el elemento por un id establecido. En este ejemplo estariamos seleccionando a un elemento con id="titulo".

Selector de Clase

Descripción

La sintaxis de esta es poner un punto antes del nombre de la clase. De esta forma seleccionamos todos los elementos que tengan esta clase en el atributo class. En este ejemplo estariamos seleccionando todos los elementos que tengan la clase container.

Ejemplo

Selector de atributo

Descripción

La sintaxis de esta es poner el elemnto y entre corchetes el atributo por el cual se quiere seleccionar.En este ejemplo estariamos seleccionando todos los elementos <a> que tengan un href y cominecen con #.

Selector de Pseudoclase

Descripción

Las pseudoclases hacen referencia a un estado de un elmento. La sintaxis de esta es poner dos puntos despues de haber seleccionado al elemento y despues la pseudoclase.En este ejemplo estariamos indicando que los estilos se aplicaran a un elemento con la clase container pero los estilos solo surtirán efecto al pasar el ratón sobre el elemento.

Ejemplo

Selector de Descendiente

Descripción

La sintaxis de esta es poner un ESAPCIO entre el elmento padre y el descendiente. De esta forma seleccionamos todos los elementos que se encuentren dentro del elemento padre. En este ejemplo estariamos seleccionando a todos los h2 que estan dentro de un div.

Box Shadow

Descripción

Es la propiedad que nos permite añadir sombras alrededor de un elemento. se pueden poner varias sombras separandolas con una coma. El primer valor es el Offset del eje X si es un valor positivo la sombra se moverá a al derecha , si es negativo ira a la izquierda . El segundo valor es el Offset en el eje Y , si se el da un valor positivo la sombra irá hacia abajo , para hacer la sombra hacia arriba basta con poner este valor negativo , despues le sigue el blur-radius y con este le indicamos que tan borrosa sera la sombra, por último tenemos el color de la sombra.
